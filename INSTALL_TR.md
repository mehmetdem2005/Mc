# Kurulum — AutoNPC PlayerAI v2.0

1. GitHub repo sayfasından `releases/AutoNPC_PlayerAI_v2_0_AllInOne.mcaddon` dosyasını indir.
2. Dosyayı telefonda Minecraft ile aç.
3. Dünya ayarlarında şu iki paketi aktif et:
   - AutoNPC PlayerAI v2.0 BP
   - AutoNPC PlayerAI v2.0 RP
4. Oyuna gir.
5. Kitap gelmezse chat'e `npc kitap` yaz.
6. İlk test için:

```text
npc test
npc işçi
npc yürü
npc odun
```

Eski AutoNPC sürümleri aktif kalırsa çakışabilir. Dünya ayarlarından eski sürümleri kapat.
